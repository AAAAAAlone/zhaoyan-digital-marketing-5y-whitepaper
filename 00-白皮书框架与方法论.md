# 赵岩的数字营销实战群 · 五年白皮书

## 编制框架（要什么 → 如何解决 → 具体方法）

### 一、我们要什么

把 **8 个「数字营销实战」群** 视为 **一个组织的多个分部**（1–7 班 + 地图联创），产出：

**《赵岩的数字营销实战群 · 五年汇报白皮书 — by 加玮》**

读者：赵岩、核心学员、合作方。  
用途：复盘组织发展、对外展示社群价值、指导下一阶段运营（GEO 时代）。

**必须回答的问题：**

1. 五年里聊了多少？何时最活跃？  
2. 整体在聊什么方向？如何演变？  
3. 谁在说？谁沉默？各自关注什么？  
4. 各内容方向下有哪些典型讨论？  
5. 分享了多少链接/资料？知识如何沉淀？

---

### 二、如何解决（三层架构）

```
┌─────────────────────────────────────────┐
│  L1  叙事层：白皮书六篇（给人读）          │
├─────────────────────────────────────────┤
│  L2  分析层：话题段 + 主题图 + 人物归一    │
├─────────────────────────────────────────┤
│  L3  数据层：wx-cli 全量 → JSONL 缓存     │
└─────────────────────────────────────────┘
```

| 篇章 | 对应问题 | 主要手段 |
|------|----------|----------|
| 第一篇 组织全景 | 规模、结构、仪表盘 | 计数、占比、参与层级 |
| 第二篇 趋势演变 | 时间、主题迁移 | 年/月聚合、话题段×年 |
| 第三篇 参与者图谱 | 谁、关注什么 | 昵称归一、人物×主题矩阵 |
| 第四篇 内容方向 | 聊什么、例证 | 线程级主题 + 代表段 |
| 第五篇 链接传播 | 资料分享 | 链接/文件消息统计 |
| 第六篇 结论建议 | So what | 编辑归纳 + 半年迭代机制 |

---

### 三、具体方法（工具箱）

| 方法 | 作用 | 来源 |
|------|------|------|
| **全量拉取 + 去重** | 避免分页重叠丢/重 | wx-cli `history --offset` |
| **话题切片（30min）** | 「在聊什么」的最小单元 | baoyu-wechat-summary |
| **线程级主题归类** | 避免单条关键词误判 | 正则主题图（可换 LLM 审定） |
| **TF-IDF / textrank** | 热词、代表词 | jieba |
| **人物归一** | 赵岩/加玮/始熊君不重复计数 | 规则表 |
| **按月 baoyu 精华** | 高质量例证（二期） | baoyu-wechat-summary 三轮 |
| **沉默人数** | 入群未发言 | `wx members`（待补） |
